Arquitectura del Sistema

Este sistema implementa una arquitectura de microservicios que separa la lógica de negocio en componentes independientes:

Backend (API RESTful)

  • FastAPI: Framework moderno y de alto rendimiento para la API
  • SQLAlchemy: ORM para interactuar con la base de datos
  • Servicios: Lógica de negocio encapsulada en servicios especializados
  • Integridad de datos: Implementada mediante servicios que reemplazan los triggers de la base de datos

Frontend

  • HTML, CSS, JavaScript: Interfaz de usuario responsive
  • Bootstrap: Framework CSS para diseño moderno
  • APIs REST: Comunicación con el backend mediante llamadas API
  • JavaScript puro: Sin dependencias de frameworks adicionales
Nota: Esta aplicación está diseñada para ser desplegada en Render, aprovechando su facilidad de implementación y escalabilidad.
